## [M-GAME] 게임 분석 모듈

### 핵심 루프 (Core Loop)

```
탭 → 화살표 이동 → 경로 개척 → 그리드 클리어 → 보상 → 다음 레벨
```

**세션 사이클** [추론]
1. **레벨 선택**: 현재 진행 레벨 자동 시작
2. **퍼즐 풀이**: 화살표 순서 분석 → 탭으로 이동
3. **클리어/실패**: 성공 시 보상, 실패 시 리트라이/광고
4. **진행**: 다음 레벨 또는 세션 종료

**예상 세션 길이**: 3-10분 (5-15 레벨) [추론]
**예상 일일 세션**: 2-4회 [추론]
