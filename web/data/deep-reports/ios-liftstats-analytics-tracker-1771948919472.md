## 타겟 유저

### 주요 페르소나
[추론] 데이터 기반 의사결정을 선호하는 진지한 웨이트 트레이닝 애호가들. 25-40대 남성 중심, IT/엔지니어링 직군이 주요 타겟. [추론] 운동 경력 1년 이상으로 단순 기록을 넘어선 분석 도구가 필요한 중급자 이상.

### 유저 여정
1. **발견**: 기존 운동 앱의 분석 기능 한계 체감 → "analytics" 키워드 검색
2. **활성화**: 기존 운동 데이터 임포트 → 새로운 인사이트 발견
3. **리텐션**: 주간/월간 분석 리포트를 통한 운동 계획 수립
4. **수익화**: 고급 분석 기능 필요 → 프리미엄 구독
5. **추천**: 헬스 커뮤니티에서 "데이터 덕후"들에게 추천
