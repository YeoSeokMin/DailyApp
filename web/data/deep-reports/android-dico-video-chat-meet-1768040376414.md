## 타겟 유저

### 주요 페르소나

**1. 글로벌 소통 희망자 (Primary)** [추론]
- 연령: 18-30대
- 성별: 남성 비중 높음 (업계 특성)
- 관심사: 외국 문화, 언어 교환, 새로운 친구
- 페인포인트: 오프라인에서 외국인 만나기 어려움
- 사용 시나리오: 저녁 시간 무료함 해소, 언어 연습

**2. 외로움 해소 니즈 (Secondary)** [추론]
- 연령: 20-40대
- 심리: 사회적 연결 욕구, 비대면 선호
- 페인포인트: 코로나 이후 대면 소통 감소
- 사용 시나리오: 주말/야간, 짧은 대화 원함

**3. 호기심 기반 탐색자 (Tertiary)** [추론]
- 연령: 10대 후반~20대 초반
- 심리: 새로운 경험 추구, 트렌드 민감
- 페인포인트: 일상의 지루함
- 사용 시나리오: 친구와 함께 사용, 재미 목적

### 유저 여정

```
발견 → 활성화 → 리텐션 → 수익화 → 추천
```

1. **발견**: 앱스토어 검색 "영상통화", "랜덤채팅" 또는 SNS 광고 [추론]
2. **활성화**: 가입 → 무료 코인 100개 지급 → 첫 매칭 경험 [추론]
3. **리텐션**: 매칭 품질, 흥미로운 대화, 실시간 번역 편의성 [추론]
4. **수익화**: 무료 코인 소진 → 결제 유도 팝업 → 구독/코인 구매 [추론]
5. **추천**: 재미있는 경험 → 친구에게 공유 (바이럴) [추론]

## 핵심 인사이트

### 개발자를 위한 교훈

1. **프라이버시를 차별점으로** [추론]
   - 기존 시장의 약점(프라이버시 우려)을 핵심 USP로 전환
   - "블러 시작", "녹화 차단" 같은 구체적 기능으로 신뢰 구축

2. **레드오션에서 후발주자 전략** [추론]
   - 전체 시장 공략 대신 특정 세그먼트 집중 (언어 교환, 안전 중시 사용자)
   - Azar와 정면 경쟁 피하고 니치 포지셔닝

3. **실시간 번역은 필수 기능** [확인]
   - 글로벌 소셜 앱에서 언어 장벽 해결은 핵심 가치
   - Azar의 성공 요인 중 하나

4. **고객지원 = 차별점** [추론]
   - 24/7 지원 강조로 경쟁사 대비 신뢰 우위 주장
   - 단, 실제 리뷰에서 지원 품질 불만 존재 → 약속과 실행 일치 필요

5. **초기 무료 경험 품질이 핵심** [추론]
   - 코인 시스템의 무료 체험 → 유료 전환 구조
   - 첫 경험이 좋아야 결제 의향 발생

   - 인증된 사용자끼리만 매칭되는 유료 옵션
