## 타겟 유저

### 주요 페르소나
[추론] 대학 동아리 회장/임원 (20-25세), 청년 모임 리더 (25-30세), 스타트업 동호회 운영자 (25-35세). [추론] 디지털 네이티브 세대로 앱 사용에 익숙하며, 효율성을 중시하는 성향.

### 유저 여정
1. **발견**: 동아리/모임 운영 중 기존 도구의 한계 경험 → 검색/추천
2. **활성화**: 첫 이벤트 생성 → 멤버 초대 → 통합 관리 편의성 체감
3. **리텐션**: 정기 모임마다 자동 사용 → 데이터 축적
4. **수익화**: 무료 한계 도달 → 고급 기능 필요 → 유료 전환
5. **추천**: 다른 동아리/모임 운영자에게 추천
