## [M-FINTECH] 핀테크 앱 특화 분석

### 보안 체계
- 인증 방식 (생체, PIN, 2FA)
- 데이터 암호화 수준
- 규제 준수 (전자금융거래법, PCI-DSS)

### 금융 연동
- 은행/카드사 API 연결
- 오픈뱅킹 활용
- 증권사/거래소 연동

### 핵심 기능
- 송금/결제/투자/보험 중 해당 영역
- 자산 관리/분석 기능
- 자동화 기능 (자동이체, 로보어드바이저)

### 수익 모델
- 거래 수수료
- 스프레드
- 프리미엄 구독
- 제휴/광고

### 신뢰 구축 요소
- 라이선스/인가 현황
- 예금자 보호/보험
- 고객 지원 체계

### 규제 리스크
- 현행 규제 적용 범위
- 예상 규제 변화
- 글로벌 확장 시 규제 이슈
