## 어떤 정보를 얻을 수 있나요?

### 1. 오늘의 트렌드 한눈에
- 오늘 뜨는 앱 카테고리
- 시장에서 주목받는 아이디어
- 놓치면 안 될 기회 영역

### 2. 앱별 상세 분석

매일 엄선된 10개 앱에 대해 다음 정보를 제공합니다:

| 분석 항목 | 예시 |
|----------|------|
| **핵심 아이디어** | "AI로 동전 촬영하면 가치 즉시 감정" |
| **어떤 문제를 해결하나** | 동전 수집가들이 전문가 없이 시세를 알 수 없는 문제 |
| **누가 쓰나** | 동전 수집가, 골동품상, 유산 정리하는 분 |
| **시장성** | 빈티지/수집품 시장 성장, 글로벌 확장 가능 |
| **수익화 방법** | 월 구독 $4.99 또는 건당 $0.99 |
| **투자 포인트** | 예상 개발비용, 예상 수익성 |

### 3. 8가지 점수로 한눈에 비교

각 앱은 8가지 기준으로 점수를 매겨 **한눈에 가치를 파악**할 수 있습니다:

- 참신성 (얼마나 새로운가)
- 필요성 (꼭 필요한가)
- 타이밍 (지금이 적기인가)
- 난이도 (만들기 쉬운가)
- 시장규모 (얼마나 큰 시장인가)
- 경쟁강도 (경쟁자가 많은가)
- 수익성 (돈이 되는가)
- 확장성 (더 키울 수 있는가)

→ **종합 점수 10점 만점으로 표시**
