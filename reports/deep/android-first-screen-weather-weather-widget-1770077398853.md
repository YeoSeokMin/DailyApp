### 주요 페르소나
[추론] 야외 활동 종사자 (건설업, 배달업, 농업), 40-60대 중장년층, 날씨 민감 직업군. [추론] 복잡한 앱보다 단순한 정보 확인을 선호하는 사용자층.

### 유저 여정
1. **발견**: 날씨 앱 실행 번거로움 → 검색/추천
2. **활성화**: 런처 설정 → 즉시 날씨 확인 경험
3. **리텐션**: 매일 폰 사용 시 자동 노출
4. **수익화**: 광고 노출 피로감 → 프리미엄 전환
5. **추천**: 동일한 불편 겪는 지인에게 추천
