```

```

## 핵심 인사이트

### 개발자를 위한 교훈

1. **IP 라이선스의 양면성**: 워해머 IP가 핵심 차별점이나, 개인 개발자는 진입 불가. 독자 IP 구축이나 공개 도메인 활용 고려 필요 [추론]

2. **니치 + 깊이 전략**: 대중성 대신 특정 팬덤의 깊은 니즈 충족으로 충성 유저 확보 가능. 1,300장 카드, 19개 군단의 방대함이 핵심 [확인]

3. **이벤트 공정성 설계**: Sealed Deck 포맷으로 과금 무관한 스킬 경쟁 제공 — P2W 비판 완화 전략 [확인]

4. **크로스플랫폼 필수**: 모바일+PC 동시 지원으로 유저 접근성 극대화. CCG 장르에서 특히 유효 [확인]

5. **커뮤니티 참여형 개발**: 팩션 워로 유저가 신규 콘텐츠 결정 참여 — 소유감과 리텐션 강화 [확인]

### 개선 제안

1. **F2P 경쟁력 강화** — 랭크 모드에서 덱 파워 매칭 도입하여 P2W 인식 완화 — 실현성: 중간

2. **세계관 온보딩 개선** — 비팬 유저를 위한 호루스 헤러시 스토리 인터랙티브 튜토리얼 — 실현성: 높음

3. **소셜 기능 강화** — 친구 간 덱 공유, 관전 모드, 스트리머 통합 — 실현성: 중간

4. **오프라인 모드** — AI 대전 오프라인 지원으로 접근성 향상 — 실현성: 높음

### 스핀오프 앱 아이디어

1. **Lore Compendium App**: 호루스 헤러시 세계관 백과사전 앱 — 카드 게임과 연동, 수집한 카드의 배경 스토리 제공 [추론]

2. **Deck Builder Companion**: 외부 덱 빌더 앱 — 메타 분석, 덱 공유, 커뮤니티 추천 기능 [추론]

3. **30K Turn-Based Tactics**: 같은 30K 세계관의 XCOM 스타일 턴제 전술 게임 — Tacticus(40K)의 30K 버전 [추론]
