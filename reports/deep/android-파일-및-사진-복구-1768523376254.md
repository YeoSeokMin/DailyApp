## 타겟 유저

### 주요 페르소나

**페르소나 1: 실수형 일반 사용자** [추론]
- **인구통계**: 18-45세, 성별 무관, 스마트폰 사진을 자주 찍는 사용자
- **심리통계**: 기술에 익숙하지 않음, 간단한 해결책 선호, 소중한 추억 보존 욕구
- **상황**: 갤러리 정리 중 실수로 중요한 사진/영상 삭제

**페르소나 2: 긴급 복구 필요 사용자** [추론]
- **인구통계**: 20-50대, 직장인/학생
- **심리통계**: 급한 상황에서 즉시 해결 원함, 비용 민감도 낮음 (긴급 시)
- **상황**: 업무용 문서, 과제 파일 등 급하게 복구해야 하는 상황

**페르소나 3: 정기적 정리 사용자** [추측]
- **인구통계**: 30-50대
- **심리통계**: 저장 공간 관리에 관심, 삭제 전 백업 습관 없음
- **상황**: 저장 공간 확보 위해 정리 후 후회하는 패턴 반복

### 유저 여정

```
발견 → 활성화 → 리텐션 → 수익화 → 추천
```

1. **발견** [추론]
   - 파일 삭제 후 "삭제된 사진 복구" 검색
   - Play 스토어에서 높은 평점(4.62)과 다운로드 수 확인 후 설치
   - 경쟁앱 대비 깔끔한 UI와 높은 평점이 선택 요인

2. **활성화** [확인]
   - 앱 실행 → 스캔 탭 → 삭제된 파일 목록 확인
   - 미리보기로 복구 대상 확인
   - 3일 무료 체험 시작

3. **리텐션** [추론]
   - 일반적으로 낮은 리텐션 (필요 시에만 사용하는 유틸리티 특성)
   - 구독 지속 사용자는 소수로 예상
   - 대부분 1회성 사용 후 삭제 패턴

4. **수익화** [확인]
   - 무료 스캔 → 복구 시 구독 요구
   - 3일 체험 후 자동 결제
   - 높은 월 구독료로 단기 LTV 극대화

5. **추천** [추측]
   - 성공적 복구 경험 시 지인 추천
   - 그러나 높은 가격으로 인한 추천 제약
